# ⚖️ 法务审核系统 v1.0

局域网内部署的AI法务审核系统，支持法律咨询、合同审核、工单流转、知识库管理。

## 功能特性

- **法务咨询** — AI驱动的法律问答，依托本地知识库精准回复
- **合同审核** — 上传合同自动分析风险点位，生成标注版Word报告
- **工单系统** — 客服收单→派单→审核→归档，全流程任务流转
- **知识库管理** — 法律文档上传自动向量化，实时增量更新索引
- **组织架构** — 树形公司→部门→人员管理，注册强制关联
- **五重鉴权** — 账号密码+JWT+加密Cookie+设备指纹+UID绑定
- **游客模式** — 免登体验，法务咨询限10轮，其他功能需注册
- **模型同步** — 自动跟随Hermes全局LLM配置，无需单独配置

## 技术栈

- **后端**: FastAPI + SQLAlchemy + ChromaDB
- **前端**: 原生HTML/CSS/JS（无框架依赖）
- **LLM**: 自动同步Hermes全局模型配置
- **数据库**: SQLite（本地存储，数据不外泄）
- **向量库**: ChromaDB（本地法律知识库）
